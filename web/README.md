# Ponte Web

React PWA-клиент для Ponte — устанавливается на iPhone через Safari «На экран Домой».

## Стек

- **React 19** с TypeScript
- **Vite** сборщик
- **Zustand** управление состоянием (персистенция в sessionStorage)
- **Socket.IO** клиент для событий в реальном времени
- **ky** HTTP-клиент с auth-интерцептором и авто-обновлением при 401
- **Framer Motion** анимации
- **WebRTC** для голосовых звонков (экспериментально)

## Возможности

- SMS-диалоги с доставкой в реальном времени
- Журнал звонков, номеронабиратель, UI входящего звонка
- Список контактов с алфавитной прокруткой и поиском
- Пересылка Android-уведомлений
- Выбор SIM-карты и доп. номера при отправке
- Автоизвлечение и копирование OTP-кодов
- Push-уведомления через service worker
- Жесты pull-to-refresh, swipe-to-delete
- Дизайн Vision Pro glassmorphism (тёмная/светлая тема)

## Установка

```bash
cp .env.example .env
# Отредактируй: VITE_API_BASE_URL, VITE_WS_URL

pnpm install
pnpm dev
```

## Переменные окружения

| Переменная              | Описание                         | По умолчанию             |
|-------------------------|----------------------------------|--------------------------|
| `VITE_API_BASE_URL`     | Базовый URL REST API бэкенда     | `http://localhost:3000`  |
| `VITE_WS_URL`           | URL WebSocket бэкенда            | `http://localhost:3000`  |
| `VITE_VAPID_PUBLIC_KEY` | VAPID-ключ для push-уведомлений  | —                        |

## Сборка

```bash
pnpm build   # выходная папка dist/
```

## Установка PWA (iOS)

1. Открой в Safari
2. Поделиться → На экран «Домой»
3. Приложение работает в полноэкранном режиме с push-уведомлениями
